Petzee is a full-stack web application that provides a centralized ecosystem for pet owners to manage their pets and access verified pet services.

Problem Statement

Pet owners currently face multiple problems:

No centralized place to manage pet medical records
Hard to find trusted groomers, walkers, or boarding centers
No vaccination or medication reminders
Difficulty contacting veterinarians quickly
Petzee solves this by providing a one-stop digital platform.



Target Users:

Pet Owners
Pet Service Providers (groomers, walkers, boarding centers)
Veterinarians
Admin




Core Features:

Pet Owner

    Register/Login
    Add multiple pets
    Upload vaccination records
    Book services
    Chat with veterinarians
    Receive vaccination reminders
    Track booking history

Service Provider

    Register & verification
    Add services and pricing
    Accept/reject bookings
    Manage schedule

Veterinarian

    Profile creation
    Online consultation chat
    Prescriptions

Admin

    Verify providers
    Manage users
    Moderate reports



Tech Stack (Proposed)

Frontend: React 
Backend: Node.js (Express)
Database: PostgreSQL
Authentication: JWT
Notifications: Email + Reminder system



System Scope

Petzee will be a service-oriented application with separate modules:

Authentication Module

Pet Management Module

Booking Module

Consultation Module

Notification Module
